about

I'm Ifeanyi Iheme Michael, a passionate Front-end Engineer based in Lagos, Nigeria. My goal is to build beautiful web applications using quality code and a user-centric design approach.

After graduating from Obafemi Awolowo University, I've dedicated my career to front-end development, honing my problem-solving skills through various collaborations, open-source projects, and personal endeavors.

I specialize in modern web technologies including React and associated libraries, Sass, Tailwind CSS, and CSS Modules etc. I'm proficient in API integration, which allows me to create dynamic and user-friendly web experiences. My skills are up-to-date with the latest web development trends, ensuring I can deliver high-quality solutions.

I hold professional certifications in Front-end Development and Data Analytics from Google, Meta, and the University of Michigan, all acquired through Coursera, Udemy and verified on Credly. I've also gained practical experience through an internship, and I remain committed to continuous learning.

Beyond coding, I enjoy diverse interests, including reading, music composition and arrangement, playing the guitar, and volunteering. I thrive in a team environment and am enthusiastic about contributing my skills effectively in any given role.
